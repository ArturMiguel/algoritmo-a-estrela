# Algoritmo A*
Trabalho para a discipina de Inteligência Artificial aplicando o algoritmo A* (*A star*) em um jogo de busca baseado na serie de jogos *The Legend of Zelda*.

### Linguagens 
> HTML 5

> CSS 3

> JavaScript

> Python 3
